Compress clinical text without flipping
a negation.

clinicalcompress protects negations, values, allergies, laterality, and temporal markers before compressing anything else — then deterministically verifies nothing clinically load-bearing was lost. No API key required.

"Patient denies chest pain."
→ naive compressor drops "denies" →
"Patient chest pain."

The problem

Generic text compressors and summarizers routinely strip short, low-frequency words to save tokens — words like "no", "not", and "denies". In clinical text, those words are not filler. Dropping one can turn an absent symptom into a present one, silently, with no error and no warning.

How protection works

Negation

"denies", "no", "ruled out", "without", "negative for"

Value

"BP 120/80", "HR 88 bpm", "10 mg", lab values

Allergy

"Allergic to penicillin", "NKDA"

Laterality

"left knee", "bilateral", "R) shoulder"

Temporal

"history of", "chronic", "resolved", "active"

Every span in these categories is located before compression runs and passes through untouched — compression only ever removes redundancy from the text around them.

The verification guarantee

  • negation_preserved — every negation cue survives, still governing the same term
  • values_intact — every numeric value and unit is unchanged
  • allergies_preserved — every allergy statement is present
  • laterality_preserved — left/right/bilateral markers survive with their referent
  • no_meaning_reversal — a negated term can never appear as an unqualified positive

If any check fails, the strict orchestrator reverts to source text and re-verifies. With strict=True (the default), the result can never contain a lost negation or a reversed clinical meaning.

Install

pip install clinicalcompress
from clinicalcompress import compress

result = compress("Patient denies chest pain. No history of MI. BP 120/80.")
print(result.compressed_text)
print(result.safety.all_passed)
clinicalcompress is a text-compression utility, not a medical device. It is not clinically validated and must not be used as the sole basis for clinical decisions. Always review compressed output before use.
